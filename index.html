About the Mandelbrot Set

The Mandelbrot Set is a set of points on the complex plane that are defined by iterating a function on each point to determine if it is in the set. To iterate the function on a given point C, set Z equal to C. The function is as follows:

Z -> Z^2 + C

Notice that C stays the same, while Z changes. Iterate this function until Z leaves the circle whose center is the origin and whose radius is 2. If Z never leaves the circle, then C is in the set, otherwise it is not in the set. No program can iterate infinitely, so this program maxes out at a default of 2000 iterations. Points in the set are colored black, while points not in the set are colored based on the number of iterations it took for that point to leave the circle. Julia sets are like the mandelbrot set, but instead of adding each unique C, you add the same value of C to every iteration point. Because of this, there is a julia set for every point on the mandelbrot set. To view the julia set for the point you are focused on, press j. The j key also toggles back to the mandelbrot set view.
